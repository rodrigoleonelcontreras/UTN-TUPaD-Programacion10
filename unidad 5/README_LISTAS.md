# TP 5 - Listas – README

Este proyecto contiene las soluciones del **Práctico 5: Listas** de Programación 1.

## Contenido

- Ejercicio 1: Lista de notas, promedio, máximo y mínimo.
- Ejercicio 2: Carga de productos, ordenamiento y eliminación.
- Ejercicio 3: Separar números pares e impares.
- Ejercicio 4: Eliminar elementos repetidos.
- Ejercicio 5: Gestión de lista de estudiantes (agregar/eliminar).
- Ejercicio 6: Rotar una lista a la derecha.
- Ejercicio 7: Matriz de temperaturas, promedios y amplitud térmica.
- Ejercicio 8: Matriz de notas, promedio por estudiante y materia.
- Ejercicio 9: Tablero Ta-Te-Ti interactivo.
- Ejercicio 10: Matriz de ventas, totales y análisis.

## Ejecución

Ejecutar el archivo en consola:

```
python tp_listas.py
```

Cada ejercicio está implementado como una función independiente.
