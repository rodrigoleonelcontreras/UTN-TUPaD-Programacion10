# TP Estructuras Repetitivas – README

Este proyecto contiene las soluciones del práctico sobre **estructuras repetitivas**.

## Contenido

- Ejercicio 1: Imprimir números del 0 al 100.
- Ejercicio 2: Contar dígitos de un número.
- Ejercicio 3: Suma entre dos valores.
- Ejercicio 4: Suma secuencial hasta ingresar 0.
- Ejercicio 5: Juego de adivinar número.
- Ejercicio 6: Números pares del 100 al 0.
- Ejercicio 7: Suma desde 0 hasta un número.
- Ejercicio 8: Clasificación de 100 números.
- Ejercicio 9: Media de 100 valores.
- Ejercicio 10: Invertir dígitos de un número.

## Ejecución

Ejecutar el archivo:

```
python tp_repetitivas.py
```

Cada ejercicio está implementado como una función distinta.
